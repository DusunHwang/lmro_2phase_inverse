### 2-1. 호출 흐름

```

 └─ default_truth()                       → LiteratureBackedTruth (frac, D, R, ...)
 └─ monotonic_native_ocps(truth, v_low, v_high)
    └─ TanhOCPParams(b0, b1, amps, centers, widths)  × R3m, C2m
    └─ make_tanh_ocp_numpy(params)         → numpy callable U(sto)
    └─ make_tanh_ocp_pybamm(params)        → pybamm Interpolant callable
 └─ TwoPhaseOCPParams(frac_R3m, U_R3m, U_C2m, D_R3m, D_C2m, R_R3m, R_C2m)
 └─ make_effective_ocp(two_phase)          → U_eff(sto) = frac_R3m·U_R3m + frac_C2m·U_C2m
 └─ make_effective_diffusivity(two_phase)  → D_eff = frac_R3m·D_R3m + frac_C2m·D_C2m
 └─ make_effective_radius(two_phase)       → R_eff = frac_R3m·R_R3m + frac_C2m·R_C2m

    └─ pybamm.Simulation(model, parameter_values=params, experiment=experiment)
    └─ sim.solve()  → time_s, voltage_v, current_a
```

### 3-2. OCP 구성 함수 (두 종류)

#### ① `_plateau_ocp()` — plateau 타입

```python
# scripts/generate_toyo_native_2phase_sample.py

def _plateau_ocp(name, voltage_low, voltage_high,
                  peak_voltage, plateau_width, plateau_depth=0.72):
    """
    dU/ds = 1 - depth × exp(-0.5·((s - target)/width)²)
    위의 미분을 적분하여 단조 증가 s(u) → V(sto) 변환
    s = 1 - sto (방전 진행도)
    peak_voltage에서 dQ/dV 피크 발생
    """
    s = np.linspace(0, 1, n)
    target = (peak_voltage - voltage_low) / (voltage_high - voltage_low)
    duds = 1 - depth × exp(Gaussian(s, target, width)) + edge_terms
    u = voltage_low + (voltage_high - voltage_low) × cumsum(duds) / total
    return InterpOCP(sto=1-s, voltage=u)
```

#### ② `_gaussian_redox_ocp()` — Gaussian redox 타입

```python
# scripts/generate_toyo_native_2phase_sample.py

def _gaussian_redox_ocp(name, voltage_low, voltage_high,
                         center_voltage, sigma_voltage,
                         peak_weight=1.0, baseline_weight=0.05):
    """
    dQ/dV 가 voltage 축에서 Gaussian 분포를 갖도록 OCP 설계:
    ds/du = baseline + peak_weight × exp(-0.5·((u - center)/sigma)²)
    u: 전압 축에서 적분 → s(u) 구성 → sto = 1 - s → V(sto) 역산
    """
    u = np.linspace(voltage_low, voltage_high, n)
    redox_density = baseline + peak × exp(Gaussian(u, center, sigma))
    s = cumsum(redox_density × du)  # s: 방전 진행도
    s = s / s[-1]
    return InterpOCP(sto=1-s, voltage=u)
```

두 함수 모두 `InterpOCP` 객체를 반환하며, `InterpOCP.pybamm(sto)` 메서드가 `pybamm.Interpolant(sto_grid, v_grid, sto, interpolator="cubic")`를 반환한다.
